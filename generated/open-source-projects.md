# Greek open source on GitHub

← [readme.md](readme.md)

Curated **open source** repositories on GitHub that are especially relevant in Greece—APIs for public services,
translations, and tooling. Use them in your work or **contribute** issues, code, or docs.

---

## Ergani Python SDK

**Repository:** [https://github.com/withlogicco/ergani-python-sdk](https://github.com/withlogicco/ergani-python-sdk)

Python SDK for the Greek government **Ergani** API (employee workflows). Useful if you integrate with Ergani from Python; check the repo for contribution guidelines.

---

## AADE myDATA (PHP)

**Repository:** [https://github.com/firebed/aade-mydata](https://github.com/firebed/aade-mydata)

PHP package for **ΑΑΔΕ myDATA** invoicing REST API—sending, cancelling, and requesting invoices, plus related methods. MIT-licensed; see the repository for docs and how to contribute.

---

## Greek Python documentation (python-docs-el)

**Repository:** [https://github.com/python/python-docs-el/](https://github.com/python/python-docs-el/)

Official **Python documentation** translated into Greek—collaboration via issues and pull requests on GitHub, coordinated with the wider translation effort.
